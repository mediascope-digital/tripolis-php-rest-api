Tripolis PHP API
================

A simple Object Oriented wrapper for Tripolis API, written with PHP5.

Requirements
------------

* PHP >= 5.5
* [Guzzle](https://github.com/guzzle/guzzle) library,
* (optional) PHPUnit to run tests.
